# Definition and Purpose
A `Package` represents the *visualization* of a `Module` or a collection of `Modules`.  In other words, it is a partular **UI** and accompanying flux mechanics.

## Example
```
/packages
  /NAME
    main.js
    /components
    /views
    <!-- The UI for the 4 Core Modes -->
    /record
      /read
        /components
      /write
        /components
    /repository
	  /components
    /schema
	  /components
```
